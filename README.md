# ПРОФ СТОМ — Информационная система стоматологической клиники

Единая информационная система для стоматологической клиники **ПРОФ СТОМ**, объединяющая три модуля:

- **Модуль администратора** (Вилков Сергей)
- **Модуль пациента** (Семёнова Александра)
- **Модуль врача-стоматолога** (Першин Егор)

## Технологии
- **Frontend**: React 18 + Vite + Material-UI + FullCalendar + Socket.io-client
- **Backend**: Node.js + Express + Sequelize + PostgreSQL + Socket.io + JWT
- **База данных**: PostgreSQL

## Возможности системы (реализованы в MVP)

### Общие
- Единая авторизация с ролями (admin, doctor, patient)
- JWT + защищённые маршруты
- Real-time обновления через WebSocket (Socket.io)
- Адаптивный интерфейс (включая планшеты)

### Модуль Администратора
- Календарь расписания (день/неделя/месяц) с FullCalendar
- Создание, подтверждение, отмена, перенос записей
- Управление пациентами, врачами, услугами
- Формирование отчётов (Excel/PDF — заглушки)
- Автоматические напоминания (симуляция)

### Модуль Пациента
- Онлайн-запись в 5 шагов (врач → услуга → дата → время → подтверждение)
- Просмотр своих записей и истории визитов
- Личный кабинет
- Отмена/перенос своих записей

### Модуль Врача
- Электронная карта пациента
- **Интерактивная зубная формула** (SVG, 32 зуба, кликабельная, с модальным редактированием статуса)
- Назначение лечения с библиотекой шаблонов
- Просмотр расписания и списка пациентов
- Загрузка документов/рентгенов (заглушка)

## Быстрый старт (локальная разработка)

### 1. Установите зависимости
```bash
# В корне проекта
cd backend && npm install
cd ../frontend && npm install
```

### 2. Настройте PostgreSQL
1. Установите PostgreSQL (локально или Docker).
2. Создайте базу данных:
   ```sql
   CREATE DATABASE prof_stom;
   ```
3. Создайте файл `backend/.env` (см. пример ниже).

### 3. Инициализируйте БД и заполните тестовыми данными
```bash
cd backend
npm run db:init
npm run db:seed
```

### 4. Запустите backend и frontend
```bash
# Терминал 1
cd backend
npm run dev

# Терминал 2
cd frontend
npm run dev
```

Приложение будет доступно:
- Frontend: http://localhost:5173
- Backend API: http://localhost:5000
